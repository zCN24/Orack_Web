## 食用教程 💡

请在浏览器访问 https://orack.asia/ 来访问官网<br>

1. 若官网无法访问
```
//解决方案

请检查您的网络质量
查看网址是否正确
将DNS更改为8.8.8.8
```

2. 如何进入游戏

```
CSGO服务器：
点击相应服务器按钮后等待游戏自动开启

Minecraft服务器：
将相应IP复制至游戏内并开始游戏
```
